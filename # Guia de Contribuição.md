# Guia de Contribuição

Obrigado por considerar contribuir para o Assistente Inteligente de Organização de Estudos (AIOE)! 🎓

## Como Contribuir

### Reportando Bugs

Antes de criar um relatório de bug, verifique a lista de issues, pois você pode descobrir que o erro já foi reportado. Quando você estiver criando um bug report, inclua o máximo de detalhes possível:

- **Use um título descritivo**
- **Descreva os passos exatos** para reproduzir o problema
- **Forneça exemplos específicos** para demonstrar os passos
- **Descreva o comportamento observado** e o que você esperava
- **Incluir screenshots e/ou gifs** se possível
- **Mencione sua versão do Python** e sistema operacional

### Sugerindo Enhancements

Sugestões de enhancements são rastreadas como issues. Ao sugerir um enhancement, inclua:

- **Use um título descritivo**
- **Forneça uma descrição passo a passo** da sugestão
- **Forneça exemplos específicos** para demonstrar os passos
- **Descreva o comportamento atual** e o comportamento sugerido
- **Explique por que esse enhancement seria útil**

### Pull Requests

- Preencha o template de PR completamente
- Siga os padrões Python PEP 8
- Inclua testes apropriados
- Atualize a documentação conforme necessário
- End all files with a newline
